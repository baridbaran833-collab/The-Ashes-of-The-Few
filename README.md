# Ashes of the Few

An original interactive neo-noir crime-thriller story universe created by Barid.

## Overview

Ashes of the Few is a cinematic web-based storytelling project that combines graphic-novel-inspired presentation, worldbuilding, and interactive web design.

Set in Los Angeles, 2026, the story explores themes of justice, vengeance, trauma, corruption, redemption, and moral ambiguity through three central characters:

* **Maxton** – a forensic laboratory owner who secretly operates as a serial killer guided by a strict code focused on protecting innocents.
* **Kane Grey** – a young self-made billionaire who believes broken systems should be improved rather than bypassed.
* **Scarper** – a psychologically scarred criminal mastermind determined to expose society's hypocrisy.

The project presents the story through a cinematic website experience featuring immersive visuals, atmosphere-driven design, character dossiers, worldbuilding elements, and narrative storytelling.
